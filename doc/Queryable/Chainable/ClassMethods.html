- (Object) chain(*names)

Makes an existing method chainable by storing its return value as the internal query, and returning the query object itself.

Examples:

chain :order_by_name

chain def search(field_values)
  field_values.inject(query) { |query, (field, value)|
    query.where(field => /#{value}/i)
  }
